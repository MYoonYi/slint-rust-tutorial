# Slint Rust Tutorial Projekt

Mein erster Versuch, GUI mit _slint_ zu entwickeln.

## About

Dieses GUI beziehungsweise dieses Spiel wird anhand des [Tutorials](https://docs.slint.dev/latest/docs/slint/tutorial/quickstart/) auf der slint-Webseite entwickelt.

## How to Play

1. Es gibt 16 Kacheln mit 8 unterschiedlichen Bilder, und jedes Bild hat ein Paar.
2. Wenn man eine Kachel klickt, wird sie umgedreht.
3. Man kann nur gleichzeitig bis zu **2** umdrehen.
4. Wenn die 2 umgedrehte Kacheln identisch sind, dann hat man das Paar aufgedeckt, und sie bleiben umgedreht.
5. Man soll alle Paare finden.
